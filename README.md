# John Hartquist

Hi, I'm John, a software engineer building at the intersection of AI, ML,
and systems. I've spent most of my career at startups, most recently as CTO
at TakeOne. My MS thesis at Cal Poly was on
[real-time polyphonic guitar transcription](https://www.youtube.com/watch?v=nt_kMxCK78I).

I'm at my best doing applied research: prototyping, training models, running
experiments, and implementing papers.

Recent personal projects include [real-time audio analysis](https://github.com/jhartquist/resonators),
[porting ML models to Rust](https://github.com/jhartquist/candle-pitch), and AI
coding agent [tooling](https://github.com/jhartquist/groxide) and
[orchestration](https://github.com/jhartquist/rloop).

More work and writing
at [johnhartquist.com](https://johnhartquist.com).
